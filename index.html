ПЕРЕД УСТАНОВКОЙ

Выберите свою систему

Скачайте установщик только из закрытого чата. После установки введите персональный ключ, который выдал администратор.

Важно: один ключ активируется на одном устройстве. Не передавайте ключ и установщик другим людям.
▣WINDOWS 10 / 11

Установка на Windows

  1. Скачайте файл Soldier-VIP-Setup.exe из закрытого чата.
  2. Запустите установщик и оставьте стандартную папку установки.
  3. Откройте Soldier VIP через ярлык на рабочем столе.
  4. Вставьте лицензионный ключ и завершите мастер первого запуска.

●MACOS · M1 / M2 / M3 / M4

Установка на Mac

  1. Скачайте ZIP-архив Soldier-VIP-macOS-apple-silicon.zip.
  2. Распакуйте ZIP только на Mac, затем откройте файл .dmg.
  3. Перетащите Soldier VIP.app в папку Applications.
  4. Откройте Terminal и выполните команду ниже один раз.
xattr -dr com.apple.quarantine "/Applications/Soldier VIP.app"
